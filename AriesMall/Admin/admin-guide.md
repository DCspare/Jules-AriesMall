The Product Manager is where you control the catalog of items available on the store.

### 1. Adding a New Product
1.  Navigate to **Product Manager** -> **Products** tab.
2.  Click the **"Add New Product"** button.
3.  **Fill in the Product Details:**
    *   **Name & Brand:** The clear title and manufacturer of the item.
    *   **Price:** Enter the amount in Rupees (‚Çπ).
    *   **Category:** Type a category (e.g., "Sneakers", "Electronics"). *Note: New categories are automatically created on the main site menu.*
    *   **Rating:** A number between 1.0 and 5.0.
4.  **Images:**
    *   **Main Image:** Paste the URL of the primary image (use the Media Hub to generate this).
    *   **Gallery:** (Optional) Paste URLs separated by commas for extra angles.
5.  **Features:** List key selling points separated by commas (e.g., "Wireless, 10hr Battery, Waterproof").
6.  Click **"Save Product"**. The website updates instantly.

### 2. Editing & Deleting
*   **Edit:** Click the **Pencil Icon** on any product card to modify its price, stock, or details.
*   **Delete:** Click the **Trash Icon** to permanently remove an item from the store.

## üñºÔ∏è Hero Slider Manager

The "Main Site Slides" tab inside the Product Manager controls the large banners on the home page.

### Creating a Professional Slide
1.  Click the **"Main Site Slides"** tab.
2.  Click **"Add New Slide"**.
3.  **Choose Your Style:**
    *   **Style A (Text Overlay):** Upload a clean background photo. Enter a "Title" and "Description". Set "Show Dark Overlay" to **Yes** so the text is readable.
    *   **Style B (Flyer/Poster):** Upload a designed graphic that already has text on it. Leave "Title" and "Description" **blank**. Set "Show Dark Overlay" to **No**.
4.  **Device Optimization:**
    *   **Desktop Image:** Use a wide landscape image (16:9 ratio).
    *   **Mobile Image:** Use a tall portrait image (9:16 ratio) so it looks great on phones.
5.  **Link:** Where should the "Shop Now" button go? (e.g., `#/category/shoes`).
6.  Set Status to **Active** and click Save.

## üé® Media Hub (Images & AI)

The Media Hub is your tool for creating high-quality visuals without needing Photoshop.

### 1. Image Enhancer (AI Fixer)
Use this tool to fix blurry, pixelated, or low-quality product photos.
1.  **Upload:** Drag & drop your image or paste a URL.
2.  **Start:** Click **"Start Batch Enhancement"**.
3.  **Wait:** The AI will sharpen details, remove blur, and fix faces.
4.  **Result:** Once done, click the **Copy Link** button. You can now paste this high-quality link into the Product Manager.

### 2. Image Upscaler (Resolution Booster)
Use this for Hero Banners that need to be huge (4K) without losing quality.
1.  **Upload:** Drop your banner image.
2.  **Scale:** Choose **2x** (usually sufficient) or **4x**.
3.  **Start:** Click **"Start Batch Upscaling"**.
4.  **Result:** Use the generated link for your "Desktop Slider" images.

### 3. Smart Uploader (Hosting)
Use this for images that are already good quality and just need to be hosted online.
1.  **Add File:** Select a file or paste a URL.
2.  **Rename (Optional):** Give it a clear name (e.g., `iphone-15-black`) for easier finding later.
3.  **Upload:** Click the cloud icon.
4.  **Copy Links:** The tool provides optimized links:
    *   Use **"Product Image"** (Square) for products.
    *   Use **"Desktop/Mobile Slider"** for banners.
